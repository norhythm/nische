# Favicon Design Spec

## Overview

ロゴ完成までの暫定faviconを作成する。白背景にBarlowフォントの "TK" をアウトライン化した `<path>` で配置。合わなければ白一色にフォールバック。

## 仕様

### アプローチB: 白背景 + "TK" (path アウトライン)

- **形式**: SVG
- **サイズ**: 32x32 viewBox
- **背景**: 白 (`#ffffff`)
- **文字**: "TK" — Barlow Medium のグリフを `<path d="...">` でアウトライン化
- **文字色**: サイトのforeground色 `rgb(60, 55, 50)` = `#3C3732`
- **配置**: 水平・垂直ともにセンター

### フォールバック（アプローチA）

ユーザーが雰囲気に合わないと判断した場合、`<path>` を削除して白一色にする。

## ファイル構成

- `app/icon.svg` — Next.js App Router が自動的に favicon として配信
- Apple Touch Icon は必要に応じて後で追加

## 技術的な注意点

- Barlowフォントのテキストレンダリングではなく、`<path>` 要素でグリフ形状を直接記述する（フォント埋め込み不要、環境依存なし）
- SVGなので任意のサイズでシャープに表示される
- ロゴ到着後はこのファイルを差し替えるだけ

## フォールバック時の変更

"TK" の `<path>` 要素を削除し、白い `<rect>` のみ残す。
