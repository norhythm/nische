# nische

## Tech Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- PagesCMS via GitHub

## Getting Started

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build

# Start production server
pnpm start

# Run ESLint
pnpm lint
```

## Mail (Development)

```bash
# Start Mailpit server
pnpm mail:start

# Start Mailpit with verbose logging
pnpm mail:dev
```
